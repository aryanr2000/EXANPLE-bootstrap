My First Bootstrap Page

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components..

The .container class provides a responsive fixed width container. Containers are the most basic layout element in Bootstrap and are required when using our default grid system. Containers are used to contain, pad, and center the content within them. While containers can be nested, most layouts do not require a nested container.
